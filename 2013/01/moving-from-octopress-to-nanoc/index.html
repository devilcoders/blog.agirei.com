Moving from Octopress to Nanoc

Posted at: 2012-02-10 09:00

Finally got Nanoc working, to replace my old Octopress blog. Octopress was fine, but the structure drove me wild!

Highlighting a piece of ruby code!

def a1
  "a123"
end

Here's the code how I got this working from Rules file:

compile '/posts/*/' do
  filter :erb
  filter :redcarpet, options: {
    fenced_code_blocks: true,
    autolink: true
  }
  filter :colorize_syntax, default_colorizer: :pygmentsrb
  layout 'post'
end
